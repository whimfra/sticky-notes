# Testing Evidence

## Automated tests

Run:
```bat
python manage.py test
```

Expected result: tests complete successfully with `OK`.

## Manual testing checklist

1. Open the home page (notes list).
2. Create a note (title + content) and confirm it appears in the list.
3. Open note detail page and confirm the content is displayed.
4. Edit the note and confirm changes saved.
5. Delete the note and confirm it is removed from the list.
